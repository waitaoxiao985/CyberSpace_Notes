# AI插件
- ### 优点
  - #### 增删改查
  - #### 支持终端命令
  - #### 支持MCP扩展功能 
- ### MCP (模型上下文协议)
  - #### MCP Host
    - ##### 本地应用, IDE或云服务
  - #### MCP Client (运行在MCP Host)
    - ##### 用于连接MCP服务器, 例如基于Python mcp库编写的代码
  - #### MCP Server (运行在目标服务器)
    - ##### 文件管理, 命令执行, 数据库访问等进程
